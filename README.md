# Ejecutar en desarrollo
1. Clonar repositorio
2. Ejecutar
```
npm i / npm install
```
3. Tener Nest CLI instaladado

```
npm i @nestjs/cli
```

4. Levantar la base de datos

```
docker-compose up -d
```


## Stack usado

* MongoDB
* Nestjs
